wasAppEndedNormally

Indicates whether application was ended normally or crahsed It gets checked on storage intialization, if flag is set to NO, PTNStorageAppCrashedNotification gets generated. Otherwise, this flag is set to NO. Upon correct app termination this flag is set to YES.

- (BOOL)wasAppEndedNormally

Return Value

YES means application ended normally, NO means application crashed

Discussion

Indicates whether application was ended normally or crahsed It gets checked on storage intialization, if flag is set to NO, PTNStorageAppCrashedNotification gets generated. Otherwise, this flag is set to NO. Upon correct app termination this flag is set to YES.
